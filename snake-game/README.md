# Neon Snake

赛博霓虹风 Web 贪吃蛇小游戏，基于 Vite + React + TypeScript。

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物在 `dist/`。

## Coolify 部署

推荐方式：Static Site / Nixpacks。

- Install Command: `npm install`
- Build Command: `npm run build`
- Publish Directory: `dist`

如果使用 Docker 部署，可自行添加 Nginx 镜像；当前项目静态站点方式最简单稳定。

## 操作

- 方向键 / WASD：控制方向
- 空格：暂停 / 继续
- Enter：重新开始
- 移动端：使用屏幕按钮控制
